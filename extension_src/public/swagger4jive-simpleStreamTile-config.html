• swaggerLogo: (optional) This URL points to an image that will be displayed along-side the activity adn external object. If not provided, it will default to: http://swagger.io/wp-content/uploads/2016/02/swagger-logo.png
  • swaggerTitle: (required) This will be the Title of the external object. This will be searchable in the Jive.
  • swaggerVersion: (optional) If provided, it will be appended to the Title of the external object.
  • swaggerDescription: (optional) If provided it will be used to describe the external object. This will be searchable in the Jive.
  • swaggerSite: (optional) If provided, a link to view a remote site with auxillary information in the activity stream.
  • swaggerURL: (required, without swaggerJSON) If provided, the ext-object will attempt to fetch the JSON spec from the above link. If swaggerJSON is defined, this value may be ignored.
  • swaggerJSON: (required, without swaggerURL) If provided, the JSON structure passed into this value will be used as the swagger specification.
